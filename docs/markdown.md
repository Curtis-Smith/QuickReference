## Markdown

#### Header (six levels available)
```
# Head 1
## Head 2
### Head 3
```
#### Lists (2 spaces for next level)
##### Unordered
```
- Item
or 
* Item
```
##### Ordered
```
1. Item 1
1. Item 2
  1. Item 2a
  1. Item 2b
```

#### Link
```
[Text](url)
[link to this doc](/docs/markdown.md)
```

#### Link - Image
```
![Alt Text]()
![aceLink](/images/playingCards/ace_heart.png)
```
